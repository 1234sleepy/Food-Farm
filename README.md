### Installation


1. Clone the repo
   ```sh
   git clone https://github.com/1234sleepy/Food-Farm
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Docker compose
   ```sh
   docker compose up -d
   ```
5. Migration(Optional)
   ```sh
   dotnet ef migrations add (name) -p Storage -s API                 IMPORTANT!!! - Change name, before using
   ```
